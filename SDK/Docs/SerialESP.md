# Serial ESP

[English version](en/SerialESP.md)

Статус Serial backend, COM-портов и ESP32: **в разработке**.

Public SDK 0.14.0 не экспортирует Serial-функции, не открывает порты и не
содержит firmware. `#TV_BACKEND_SERIAL_ESP` зарезервирован для будущей версии;
его выбор сейчас завершается ошибкой неподдерживаемого backend.

`Example04_SerialESP.pb` является только компилируемым уведомлением о статусе и
не обращается к аппаратуре. Будущая реализация потребует отдельного выпуска и
документированной версии протокола.
