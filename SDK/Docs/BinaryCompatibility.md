# Бинарная совместимость

[English version](en/BinaryCompatibility.md)

Документированная таблица экспортов DLL составляет ABI. Внутренние символы,
исходные структуры и `TV_Internal*` не являются API.

- совместимость сохраняется в пределах major-версии;
- новые функции допустимы в minor-версии;
- изменение сигнатуры или layout публичной структуры требует major-версии либо
  новой версионированной структуры;
- DLL и include обязаны иметь одинаковые `#TV_API_VERSION`;
- приложение не должно искать или вызывать внутренние символы.

`TV_Config` начинается с `StructureSize` и `APIVersion`. `TV_Init()` проверяет
оба поля. Дополнительно доступны `TV_CheckAPIVersion()` и `TV_InitEx()`.

DLL экспортирует cdecl-функции (`ProcedureCDLL`), public include использует
`ImportC`. Числовые ID непрозрачны. Переданная приложением память остаётся его
собственностью; DLL не требует освобождать возвращённые указатели.

Строки копируются в UTF-16 буфер вызывающего функциями `TV_Copy*`. Результат —
требуемая ёмкость в символах с завершающим нулём.
