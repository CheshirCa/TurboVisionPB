# Ошибки и диагностика

Большинство функций возвращают `#False` или ID 0 при ошибке. После отказа
прочитайте `TV_GetLastError()` и скопируйте текст через
`TV_CopyLastErrorText(*Buffer, BufferCharacters)`.

Типовые причины: неверный параметр, вызов до инициализации, неподдерживаемый
backend, отсутствие настоящего console handle и слишком маленький терминал.
Сообщения public DLL не содержат исходные пути и адреса внутренних объектов.

После частично успешного `TV_Init()` допустимо вызвать `TV_Shutdown()`.
Повторная инициализация без Shutdown отвергается. Встроенный TRACE и callback
логирования в public 0.14.0 отсутствуют.
